Seata 2.1.0​

Seata 2.0.0 发布

Seata 是一款开源的分布式事务解决方案，提供高性能和简单易用的分布式事务服务。

此版本更新如下：

feature:​

  • [#6370] seata saga spring接耦、架构优化。
  • [#6205] 提供mock server
  • [#6169] 支持新版本状态机设计器
  • [#6230] 支持RocketMQ消息事务
